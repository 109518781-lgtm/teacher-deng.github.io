平面向量最值题：一步一步讲明白

适合对“爪子定理、等和线、极化恒等式”还不熟、觉得抽象的学生
先记住一句话：这类题不要一上来硬算。先看“谁固定、谁在动、能不能转成长度、能不能转成二次函数”。
很多最值题，本质上只是在做这三件事：配方、投影、找几何意义。

一、先把三个工具讲懂

1. 极化恒等式：把点积变成长度

最常用的两个式子：
a·b = (|a+b|²-|a|²-|b|²)/2
a·b = (|a|²+|b|²-|a-b|²)/2
它的作用很简单：原本看不见的“夹角信息”，转成看得见的“长度信息”。
比如知道 |a|、|b|、|a-b|，就能立刻求 a·b。

2. 等和线：|X-A|+|X-B| 的最小值

把它想成“从 A 走到 X，再从 X 走到 B”。
那总路程最短什么时候出现？当然是 X 就落在 AB 这条直线上，最短路程就是：
|X-A|+|X-B| ≥ |A-B|
如果题目写成 |a-p|+|a-q| 的最小值，本质也是同一件事：中间点动，两个端点固定。

3. “爪子定理”这里怎么理解最省事

很多老师会把“从一个动点伸出两条向量去抓两个定点”的结构叫“爪子”。
最常见形式是：
PA·PB， 或者 MN·BN， 或者 (a-b)·(b-c)
处理方法几乎都一样：
  • 先设坐标，把动点写成一个参数 t；
  • 点积展开成关于 t 的二次式；
  • 再配方，最值马上出来。
所以别怕它抽象，很多时候它就是一元二次函数。

二、逐题讲解

第1题

已知 |a|=|b|=a·b=c·(a+2b-2c)=2，求 |a-c| 的范围。
答案：A
第1步：先把 a、b 放进坐标系。
因为 |a|=|b|=2，且 a·b=2，所以
a·b=|a||b|cosθ = 4cosθ = 2
cosθ = 1/2
也就是 a、b 的夹角是 60°。不妨设
a=(2,0), b=(1,√3)
这样既满足 |a|=|b|=2，也满足 a·b=2。
第2步：把关于 c 的条件化简。
由
c·(a+2b-2c)=2
代入 a+2b=(4, 2√3)，得
(4,2√3)·c - 2|c|² = 2
2|c|² - (4,2√3)·c + 2 = 0
两边同时除以 2：
|c|² - (2,√3)·c + 1 = 0
第3步：配成“圆”的形式。
注意
|c-(1,√3/2)|² = |c|² - (2,√3)·c + 1 + 3/4
所以原式等价于
|c-(1,√3/2)|² = 3/4
这说明：点 c 落在一个圆上。
  • 圆心：O=(1, √3/2)
  • 半径：r=√3/2
第4步：题目要的是 |a-c|。
也就是：固定点 a=(2,0) 到这个圆上动点 c 的距离范围。
先算点 a 到圆心 O 的距离：
|a-O| = |(2,0)-(1,√3/2)| = √(1²+(√3/2)²)=√7/2
于是根据“点到圆上动点的距离范围”：
|a-c| ∈ [|a-O|-r, |a-O|+r]
|a-c| ∈ [(√7-√3)/2, (√7+√3)/2]
所以选 A。
这题最关键的突破口是：把条件配成圆，而不是硬算。

第2题

已知 |e|=1，b 与 e 的夹角是 60°，且 |b-te|≥|b-e| 对任意实数 t 恒成立，求 |a+2e|+|a-b| 的最小值。
答案：B = 2√3
第1步：理解条件 |b-te|≥|b-e|。
te 表示过原点、方向为 e 的一条直线上的点。
题目说：从 b 到这条直线上的所有点里，距离最小的那个恰好是 e。
也就是说，e 是 b 在这条直线上的垂足投影点。
所以 b 在 e 方向上的投影长度就是 1：
b·e = 1
又因为夹角是 60°，|e|=1：
b·e = |b||e|cos60° = |b|/2 = 1
|b| = 2
第2步：把和式最小值看成等和线。
|a+2e| = |a-(-2e)|
|a+2e|+|a-b| = |a-(-2e)|+|a-b|
这是“点 a 到两个定点 -2e、b 的距离和”。最小值就是两定点间距离：
min = |-2e-b| = |b+2e|
第3步：算 |b+2e|。
|b+2e|² = |b|²+|2e|²+2b·2e = 4+4+4(b·e)=4+4+4=12
|b+2e| = 2√3

第3题

已知 |a|=|b|=1，|c|=3，且 a·c=b·c，判断各选项。
成立的是：A、B、D
第1步：把条件 a·c=b·c 改写。
(a-b)·c = 0
所以
a-b ⟂ c
第2步：看 |a-b+c|。
|a-b+c|² = |a-b|²+|c|² = |a-b|²+9
因为 a-b 与 c 垂直，所以没有交叉项。
又因为 |a-b|∈[0,2]，所以
|a-b+c| ∈ [3, √13]
故 C 错，D 对。
第3步：看 |a+b+c|。
由于 a、b 对 c 的投影相等，所以可以让 a、b 同向，也可以反向。
  • a=b=c 方向时，|a+b+c|=1+1+3=5，故 B 对；
  • a=b 与 c 反向时，|a+b+c|=|-1-1+3|=1，故 A 对。

第4题

已知 |a|=2|b|，|b|≠0，方程 x²+|a|x+a·b=0 有实根，求 a 与 b 的夹角范围。
答案：B = [π/3, π]
第1步：方程有实根，看判别式。
Δ = |a|² - 4(a·b) ≥ 0
又因为 |a|=2|b|，所以
4|b|² - 4(|a||b|cosθ) ≥ 0
4|b|² - 4(2|b|²cosθ) ≥ 0
1 - 2cosθ ≥ 0
cosθ ≤ 1/2
θ ∈ [π/3, π]

第5题（上）

在 △ABC 中，AB=4，AC=8，∠BAC=60°，M 为 BC 中点，N 在 AC 上运动，求 MN·BN 的最小值。
答案：15/4
第1步：设坐标。
让 A=(0,0)，AC 在 x 轴上，则 C=(8,0)。
因为 AB=4，且与 AC 夹角 60°，所以
B=(4cos60°, 4sin60°)=(2, 2√3)
M 是 BC 中点：
M=((2+8)/2, (2√3+0)/2)=(5, √3)
设 N=(t,0)，其中 0≤t≤8。
第2步：写向量。
MN=(t-5, -√3)
BN=(t-2, -2√3)
第3步：点积展开。
MN·BN = (t-5)(t-2)+(-√3)(-2√3)
= t²-7t+10+6 = t²-7t+16
这就是“爪子定理”常见套路：最后就是一个二次函数。
第4步：配方。
t²-7t+16 = (t-7/2)² + 15/4
所以最小值是
15/4

第5题（下）

矩形 ABCD 中，AB=2BC=2，P 在 CD 上运动，求 PA·PB − PA·BC 的最小值。
答案：1
第1步：先把边长看清。
AB=2, BC=1
设 A=(0,0)，B=(2,0)，C=(2,1)，D=(0,1)。
P 在 CD 上，所以设 P=(x,1)，0≤x≤2。
第2步：写向量。
PA=A-P=(-x,-1)
PB=B-P=(2-x,-1)
BC=(0,1)
第3步：分别算两部分。
PA·PB = (-x)(2-x)+(-1)(-1)=x²-2x+1=(x-1)²
PA·BC = (-x,-1)·(0,1)=-1
所以原式
(x-1)²-(-1)=(x-1)²+1
最小值就是
1

第6题

在 △ABC 中，AB=AC=2，BC=2√3，P 在 BC 上运动，求 PA·PB 的最小值。
答案：B = -3/4
第1步：设坐标。
这是等腰三角形。让 B=(-√3,0)，C=(√3,0)。
则 A 在中垂线上，且
AB² = (√3)²+h² = 4
h=1
所以 A=(0,1)。
设 P=(x,0)，其中 -√3≤x≤√3。
第2步：写向量并点积。
PA=A-P=(-x,1)
PB=B-P=(-√3-x,0)
PA·PB = (-x)(-√3-x)+1·0 = x²+√3x
第3步：配方。
x²+√3x = (x+√3/2)² - 3/4
最小值为
-3/4

第7题

△ABC 为等腰直角三角形，∠C=90°，直角边长为 2。P 在平面内，且 AP 为单位向量。点 D 满足 CD=DP，求 BC·BD 的最小值。
答案：B = 3
第1步：设坐标。
取 C=(0,0)，A=(2,0)，B=(0,2)。
因为 AP 是单位向量，所以
|AP|=1
说明 P 在以 A 为圆心、1 为半径的圆上。
第2步：由 CD=DP 看出 D 是 CP 中点。
D = (C+P)/2 = P/2
设 D=(u,v)。
第3步：写点积。
BC = C-B = (0,-2)
BD = D-B = (u, v-2)
BC·BD = (0,-2)·(u,v-2) = -2(v-2)=4-2v
要让它最小，就要让 v 最大。
第4步：找 D 的最高点。
P 在圆 (x-2)²+y²=1 上，最高点是 P=(2,1)。
所以 D=P/2=(1,1/2)，此时 v 最大为 1/2。
min(BC·BD)=4-2×(1/2)=3

第8题

已知 a,b,c 都是单位向量，且 |a-b|=1，求 (a-b)·(b-c) 的最大值。
答案：B = 1/2
第1步：先用极化恒等式求 a·b。
|a-b|² = |a|²+|b|²-2a·b
1 = 1+1-2a·b
a·b = 1/2
第2步：展开原式。
(a-b)·(b-c)=a·b-a·c-|b|²+b·c
= 1/2-a·c-1+b·c
= b·c-a·c-1/2
这样看还不够直接，我们换个更聪明的方法。
第3步：把 a-b 看成一个整体 u。
设 u=a-b，则 |u|=1
(a-b)·(b-c)=u·b-u·c
先算
u·b=(a-b)·b=a·b-|b|²=1/2-1=-1/2
所以
原式 = -1/2 - u·c
要最大，就要让 u·c 最小。因为 |u|=|c|=1，所以
u·c ≥ -1
最小时取 -1，于是
max = -1/2 - (-1) = 1/2

第9题

已知 |a|=1，|b|=√2，a·b=1，|c|=2，求 (a+b)·c 的最大值。
答案：A = 2√5
第1步：点积最大值来自同向。
(a+b)·c ≤ |a+b||c|
所以关键是先算 |a+b|。
第2步：用平方公式。
|a+b|² = |a|²+|b|²+2a·b = 1+2+2 = 5
|a+b| = √5
因此
max (a+b)·c = |a+b||c| = √5×2 = 2√5

第10题（多选）

已知非零向量 a、b 的夹角为 θ，且 |a|=|2a-b|=2，判断各选项。
答案：A、C、D
第1步：把条件平方展开。
|2a-b|² = |2a|²+|b|²-2(2a)·b
4 = 16 + |b|² - 4a·b
又因为 |a|=2，a·b=|a||b|cosθ=2|b|cosθ，所以
4 = 16 + |b|² - 8|b|cosθ
|b|² - 8|b|cosθ + 12 = 0
先看 A：若 |b|=3
9 - 24cosθ +12 = 0
24cosθ=21
cosθ=7/8
所以 A 对。
再看 B：若 a∥b
这时 θ=0（反向不可能，因为会使上式无正根）。代入得
|b|² - 8|b| +12 =0
(|b|-2)(|b|-6)=0
所以 |b| 可以是 2，也可以是 6，不唯一。
故 B 错。
再看 C：求 θ 的范围。
把上式看成关于 |b| 的一元二次方程，要有实数解，判别式必须 ≥0：
Δ = (8cosθ)² - 4×12 ≥ 0
64cos²θ - 48 ≥ 0
cos²θ ≥ 3/4
又因为 |b| 的和为 8cosθ，要想有正根，必须 cosθ>0。
cosθ ≥ √3/2
θ ∈ [0, π/6]
所以 C 对。
最后看 D：a·b 的最大值。
当 θ=0，且 |b|=6 时，
a·b = |a||b|cosθ = 2×6×1 = 12
确实能取到，所以 D 对。

三、最后帮学生做一个“不会乱”的总结

看到最值题，按这个顺序想：
  1. 能不能设坐标？只要有“线段上动点”，优先设坐标。
  2. 能不能转成长度和？遇到 |x-p|+|x-q|，想到等和线。
  3. 能不能用极化恒等式？遇到 |a-b|、|a+b|、a·b 混在一起时优先用。
  4. 能不能配方？只要出现一个参数 t，点积展开后常常就是二次式。
一句最实用的话：
向量最值题，不是“技巧太神秘”，而是“你还没把它看成长度题、坐标题、二次函数题”。
一旦看成这三类，它就会突然简单很多。
